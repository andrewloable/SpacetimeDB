# SpacetimeDB Go Server SDK

The Go server SDK lets you write SpacetimeDB modules in Go. Modules are compiled
to WebAssembly with [TinyGo](https://tinygo.org) and published to a SpacetimeDB
instance.

## Requirements

- [TinyGo](https://tinygo.org/getting-started/install/) ≥ 0.33
- Go 1.23+

## Quick Start

### 1. Create a new module

```bash
spacetime init --lang go my-module
cd my-module/spacetimedb
```

This scaffolds:
```
spacetimedb/
  go.mod       # module manifest
  main.go      # example module with a Person table
  stdb.yaml    # schema for stdbgen code generation
```

### 2. Add your schema to `stdb.yaml`

```yaml
package: main
tables:
  - name: Player
    columns:
      - name: identity
        type: U64
      - name: username
        type: String
    access: public

reducers:
  - name: SetUsername
    params:
      - name: name
        type: String
    visibility: public
```

### 3. Generate registration boilerplate

```bash
go generate   # or: go run github.com/clockworklabs/spacetimedb-go-server/cmd/stdbgen
```

This creates `generated_stdb.go` with:
- Struct type definition
- BSATN encode/decode functions
- `TableHandle` variable for table operations
- `init()` with `RegisterTableDef`, `RegisterReducerDef`, `RegisterReducerHandler`
- Handler skeleton functions to fill in

### 4. Implement your reducers

Edit the generated handler skeletons in `generated_stdb.go` (or copy them to a
new file):

```go
func handleSetUsername(ctx spacetimedb.ReducerContext, args sys.BytesSource) {
    data, err := sys.ReadBytesSource(args)
    if err != nil {
        spacetimedb.LogError("SetUsername: " + err.Error())
        return
    }
    r := bsatn.NewReader(data)
    name, err := r.ReadString()
    if err != nil {
        spacetimedb.LogError("SetUsername: " + err.Error())
        return
    }
    spacetimedb.LogInfo("Hello, " + name)
}
```

### 5. Build and publish

```bash
# Compile to WASM
tinygo build -target wasm -o module.wasm ./

# Publish to a running SpacetimeDB instance
spacetime publish --module-path . my-module
```

## Manual Registration (without stdbgen)

You can register tables and reducers directly without code generation:

```go
package main

import (
    spacetimedb "github.com/clockworklabs/spacetimedb-go-server"
    "github.com/clockworklabs/spacetimedb-go/bsatn"
    "github.com/clockworklabs/spacetimedb-go/types"
    "github.com/clockworklabs/spacetimedb-go-server/sys"
)

type Person struct{ Name string }

func encodePerson(w *bsatn.Writer, p Person) { w.WriteString(p.Name) }
func decodePerson(r *bsatn.Reader) (Person, error) {
    name, err := r.ReadString()
    return Person{Name: name}, err
}

var personTable = spacetimedb.NewTableHandle("Person", encodePerson, decodePerson)

func init() {
    spacetimedb.RegisterTableDef(spacetimedb.TableDef{
        Name:    "Person",
        Columns: []spacetimedb.ColumnDef{{Name: "name", Type: types.AlgebraicString}},
        Access:  spacetimedb.TableAccessPublic,
    })
    spacetimedb.RegisterReducerDef(spacetimedb.ReducerDef{
        Name:       "Add",
        Params:     []spacetimedb.ColumnDef{{Name: "name", Type: types.AlgebraicString}},
        Visibility: spacetimedb.ReducerVisibilityClientCallable,
    })
    spacetimedb.RegisterReducerHandler(addReducer)
}

func addReducer(_ spacetimedb.ReducerContext, args sys.BytesSource) {
    data, _ := sys.ReadBytesSource(args)
    r := bsatn.NewReader(data)
    name, _ := r.ReadString()
    personTable.Insert(Person{Name: name})
}

func main() {} // required by TinyGo
```

## Table Operations

```go
// Insert a row (returns row with any auto-generated values)
row, err := personTable.Insert(Person{Name: "Alice"})

// Delete a row
deleted, err := personTable.Delete(Person{Name: "Alice"})

// Count rows
n, err := personTable.Count()

// Iterate over all rows
for person, err := range personTable.Iter() {
    if err != nil { break }
    spacetimedb.LogInfo("Person: " + person.Name)
}
```

## Unique Index Operations

```go
// Declare a unique index (usually generated by stdbgen)
var byName = spacetimedb.NewUniqueIndex[Person, string](
    "Person", "Person_name_idx",
    func(w *bsatn.Writer, name string) { w.WriteString(name) },
    encodePerson, decodePerson,
)

// Find by unique key
p, err := byName.Find("Alice")      // returns *Person or nil
deleted, err := byName.Delete("Alice")
updated, err := byName.Update(Person{Name: "Alice"})
```

## BTree Index (range scan)

```go
var byAge = spacetimedb.NewBTreeIndex[Person, uint32](
    "Person_age_idx",
    func(w *bsatn.Writer, age uint32) { w.WriteU32(age) },
    decodePerson,
)

// Iterate over all rows matching age = 30
for person, err := range byAge.Filter(uint32(30)) {
    if err != nil { break }
}
```

## Lifecycle Reducers

Register lifecycle reducers (init, on_connect, on_disconnect):

```go
func init() {
    spacetimedb.RegisterLifecycleDef(spacetimedb.LifecycleDef{
        Kind:    spacetimedb.LifecycleOnConnect,
        Reducer: "ClientConnected",
    })
}
```

## Logging

| `String`    | `string`  | `types.AlgebraicString`   |

## stdbgen Reference

`stdbgen` reads a YAML schema and generates Go registration code.

```
Usage: stdbgen [-schema stdb.yaml] [-out generated_stdb.go] [-pkg main]

Flags:
  -schema string   Path to schema YAML file (default "stdb.yaml")
  -out    string   Output Go file (default "generated_stdb.go")
  -pkg    string   Package name (default: from schema or "main")
```

Schema fields:

```yaml
package: main          # Go package name
tables:
  - name: TableName
    columns:
      - name: col_name
        type: String    # See Type Reference above
    primary_key: [col_name]   # optional
    access: public            # public or private (default: public)
    is_event: false           # boolean (default: false)
reducers:
  - name: ReducerName
    params:
      - name: param_name
        type: String
    visibility: public        # public or private (default: public)
```
